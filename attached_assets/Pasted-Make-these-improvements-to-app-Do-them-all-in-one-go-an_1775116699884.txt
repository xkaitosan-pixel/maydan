ADD 3 - Ranked Mode (تحدي المتصدرين):
═══════════════════════════════
New game mode: Real-time 1v1 Ranked Battle

How it works:
1. Player selects 3 preferred categories
2. System finds opponent (any available player)
3. Game uses randomly selected category
4. 10 questions, 10 seconds each
5. Scoring: answer in second 1 = 10pts, 
   second 2 = 9pts... second 10 = 1pt, no answer = 0pt
6. After each question: 1.5 second result screen
   showing both players' answers and points
7. Question ends immediately when BOTH players answer
8. Final screen: winner announcement + points breakdown
9. If opponent disconnects: auto-win after 30 seconds

Matchmaking:
- Player enters queue → searches for opponent
- Show "جاري البحث عن خصم..." with animation
- If no opponent found in 60s → offer Practice mode

Ranked Leaderboard:
- ONLY ranked mode scores count
- Weekly season (resets every Sunday)
- Ranks: برونز🥉 → فضة🥈 → ذهب🥇 → بلاتين💎 → أسطورة👑
- Points needed: 0/100/300/600/1000
- Show rank badge on profile

Database - run this SQL in Supabase:
CREATE TABLE IF NOT EXISTS ranked_queue (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id TEXT,
  username TEXT,
  preferred_categories JSONB,
  rank_points INT DEFAULT 0,
  status TEXT DEFAULT 'waiting',
  created_at TIMESTAMP DEFAULT now()
);

CREATE TABLE IF NOT EXISTS ranked_matches (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  player1_id TEXT,
  player1_name TEXT,
  player2_id TEXT,
  player2_name TEXT,
  category TEXT,
  status TEXT DEFAULT 'active',
  current_question INT DEFAULT 0,
  player1_score INT DEFAULT 0,
  player2_score INT DEFAULT 0,
  winner_id TEXT,
  created_at TIMESTAMP DEFAULT now()
);
