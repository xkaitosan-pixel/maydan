Add a new "Party Mode" (وضع التجمعات) to ميدان app.

HOW IT WORKS:
- Host opens app and creates a party room
- A large room code appears (e.g. "MAYDAN-4829")
- Guests join by going to maydanapp.com/party 
  and entering the room code
- No login required for guests - just enter a nickname
- Host controls the game from their device
- Questions appear on host screen (meant for TV display)
- Players answer from their phones
- Live leaderboard updates after each question
- Works with Supabase Realtime

Run this SQL in Supabase first:
CREATE TABLE IF NOT EXISTS party_rooms (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  code TEXT UNIQUE NOT NULL,
  host_id TEXT,
  status TEXT DEFAULT 'waiting',
  category TEXT,
  current_question INT DEFAULT 0,
  total_questions INT DEFAULT 10,
  created_at TIMESTAMP DEFAULT now()
);

CREATE TABLE IF NOT EXISTS party_players (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  room_code TEXT REFERENCES party_rooms(code),
  nickname TEXT,
  score INT DEFAULT 0,
  answered_current BOOLEAN DEFAULT false,
  last_answer INT,
  created_at TIMESTAMP DEFAULT now()
);
